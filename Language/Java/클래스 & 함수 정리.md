## Arrays 클래스
```
Arrays.sort(array): 오름차순 정렬
Arrays.sort(array, Collections.reverseOrder()): 내림차순 정렬 (단, 기본 타입 배열에는 적용 불가).
Arrays.copyOf(array, length): 배열 복사하여 새 배열 생성
Arrays.asList(array): 배열을 리스트로 변환
```

## List  클래스
```
📍 리스트 생성 및 초기화
Arrays.asList(array) : 배열을 리스트로 변환 (수정 불가)
new ArrayList<>(List.of(elements…)) : 수정 가능한 리스트 생성

📍 원소 추가
list.add(element) : 리스트 끝에 원소 추가
list.add(index, element) : 특정 위치에 원소 추가
list.addAll(Collection c) : 여러 개의 원소 추가
📍 원소 삭제
list.remove(index) : 특정 인덱스의 원소 삭제
list.remove(Object o) : 특정 값 삭제 (remove(Integer.valueOf(value)) 사용)
list.clear() : 리스트의 모든 원소 삭제
📍 원소 접근 및 변경
list.get(index) : 특정 인덱스의 원소 가져오기
list.set(index, element) : 특정 인덱스의 원소 변경
📍 리스트 크기 및 탐색
list.size() : 리스트 크기 반환
list.contains(element) : 특정 값 포함 여부 확인
list.indexOf(element) : 특정 값의 첫 번째 등장 위치 반환
list.lastIndexOf(element) : 특정 값의 마지막 등장 위치 반환
📍 리스트 정렬
Collections.sort(list) : 오름차순 정렬
list.sort(Comparator.reverseOrder()) : 내림차순 정렬
📍 리스트 변환
list.toArray(new Integer[0]) : 리스트 → 배열 변환
Arrays.asList(array) : 배열 → 리스트 변환 (수정 불가)
new ArrayList<>(Arrays.asList(array)) : 수정 가능한 리스트 변환
📍 리스트 반복문
for (int i = 0; i < list.size(); i++) : 일반 for문
for (int num : list) : 향상된 for문
list.forEach(System.out::println) : forEach 문법
📍 리스트 뒤집기 및 섞기
Collections.reverse(list) : 리스트 뒤집기
Collections.shuffle(list) : 리스트 랜덤 섞기
📍 리스트 비교
list.equals(otherList) : 두 리스트 비교

```

## Collections 클래스
```
Collections.sort(list): 오름차순 정렬
Collections.sort(list, Collections.reverseOrder()): 내림차 정렬
Collections.max/[min(collection): 최대/최소값 반환
Collections.reverse(list): 리스트 반전
```

## PriorityQueue 클래스 (힙 구현)
```
PriorityQueue<Integer> pq = new PriorityQueue<>(): 최소 힙
PriorityQueue<Integer> pq = new PriorityQueue<>(Collections.reverseOrder()): 최대 힙
pq.add/offer(element): 요소 추가
pq.poll(): 요소를 꺼내고 반환
pq.peek(): 요소를 조회
```

## String 클래스
```
str.length(): 길이 반환
str.charAt(index): 특정 위치의 문자 반환
str.substring(beginIndex, endIndex): 부분 문자열 반환
str.indexOf(substring): 주어진 부분 문자열이 시작되는 인덱스 반환
str.contains(substring): 특정 부분 문자열 포함 확인
str.split(delimiter): 문자열을 구분자에 따라 분할하여 배열로 반환
str.replace(oldChar, newChar): 특정 문자를 다른 문자로 교체
```

## StringBuilder 클래스
```
StringBuilder sb = new StringBuilder(): StringBuilder 객체 생성
sb.append(value): 값 추가
sb.insert(index, value): 특정 위치에 값 삽입
sb.delete(start, end): 시작 인덱스부터 끝 인덱스 전까지 삭제
sb.toString(): StringBuilder 객체를 일반 문자열로 변환
```

## List <-> Array
```
배열 -> 리스트
String[] array = {"apple", "banana", "cherry"};
List<String> list = new ArrayList<>(Arrays.asList(array));

리스트 -> 배열
List<String> list = new ArrayList<>();
String[] array = list.toArray(new String[0]);
```
